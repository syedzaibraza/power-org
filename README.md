# POWER Organization

People Organizing for Welfare, Equality & Reform тАФ A Pakistani-American humanitarian nonprofit website.

## Tech Stack

- **Next.js 15** (App Router)
- **React 18**
- **Tailwind CSS**
- **Lucide React** (icons)

Minimal dependencies for a fast, maintainable site.

## Running the project

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```
